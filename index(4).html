Описание проекта

Приложение позволяет управлять товарами, сотрудниками и продажами. Все данные хранятся в базе SQLite.

Инструкция по скачиванию

Windows

  1. Скачайте EXE.
  2. Запустите файл prilozh.exe.
  3. Если Windows предупредит о безопасности — нажмите «Подробнее» → «Выполнить в любом случае».

Android

  1. Скачайте APK.
  2. Разрешите установку из неизвестных источников (при необходимости).
  3. Установите и откройте приложение.

База данных

  • Скачайте shoe_shop.db.
  • Используйте её вместе с приложением или откройте через SQLite Browser.
